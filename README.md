# 好物推荐页

这是击歌吟的好物推荐静态页面，托管在 GitHub Pages，展示精选商品并引导用户通过淘宝联盟链接购买。

## 快速部署

```bash
chmod +x deploy.sh
./deploy.sh --token ghp_您的GitHubToken
```

没有 Token？看 `deploy.sh` 里的手动部署指引，2 分钟搞定。

## 页面更新

页面内容由 cron 任务自动更新（每日 14:00），不需要手动修改。

## 佣金链路

1. **小红书用户**看到好物推荐笔记
2. **点击你的个人主页链接**（需要你设置）
3. **跳转淘宝联盟推广页面** → 自动匹配你的 PID
4. **用户购买商品** → 佣金到你的淘宝联盟账户
5. **支付宝提现**（满 1 元可提）
